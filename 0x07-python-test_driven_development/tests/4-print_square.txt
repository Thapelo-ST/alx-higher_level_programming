============================
How to use 4-print_square.py
============================

This function is very simple to use it takes an argument and prints a square
from that given number

Importing document and module:
==============================
::
    >>> square = __import__("4-print_square").print_square

::
    >>> m = __import__("4-print_square").__doc__
    >>> len(m)>1
    True
